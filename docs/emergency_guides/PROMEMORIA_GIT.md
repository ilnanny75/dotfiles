# 📚 Promemoria Git - Laboratorio 2026

| Comando | Descrizione Semplice | Quando usarlo |
| :--- | :--- | :--- |
| **git status** | Controlla cosa hai cambiato | **Sempre**, prima di iniziare |
| **git add .** | Prepara i file per il salvataggio | Dopo aver finito una modifica |
| **git commit -m "..."** | Salva con un commento | Per "confermare" il lavoro sul PC |
| **git push** | Invia tutto su GitHub | Per aggiornare il sito online |
| **git pull --rebase** | Scarica le novità dal server | Prima di iniziare a lavorare |
| **git log --oneline** | Mostra la cronologia breve | Per vedere cosa hai fatto ieri |
| **git branch -M main** | Rinomina il ramo in "main" | Se vedi ancora scritto "master" |

---
*Nota: Ricordati di usare `pigia` per fare tutto in un colpo solo!*
