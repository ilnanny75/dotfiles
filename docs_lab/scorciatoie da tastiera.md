🚀 Guida alle Scorciatoie da Tastiera (Laptop Linux)

Questa guida è specifica per MX Linux (XFCE), Thunar e i principali software di editing e grafica.

📂 Gestione File (Thunar / Desktop)

Nota: Sui laptop, se i tasti F1-F12 controllano il volume/luminosità, premi sempre Fn insieme al tasto.
Azione  Scorciatoia
Rinomina file   F2 (oppure Fn + F2)
Rinomina file (Alternativa) Ctrl + M
Apri Terminale qui  F4 (o Fn + F4)
Cerca file  Ctrl + F
Proprietà file  Alt + Invio
Sposta nel cestino  Canc
Elimina definitivamente Shift + Canc

📝 Editor di Testo (Terminale & GUI)

Nano (Semplice, da terminale)

    Salvare: Ctrl + O (poi premi Invio)

    Uscire: Ctrl + X

    Cerca testo: Ctrl + W

    Incolla: Ctrl + U

Vim (Modale, da terminale)

    Modalità Inserimento: Premi i

    Tornare a modalità Comando: Esc

    Salvare ed uscire: :wq

    Uscire senza salvare: :q!

    Cancellare una riga: dd (in modalità comando)

Geany (Editor Grafico)

    Nuovo file: Ctrl + N

    Commenta riga: Ctrl + E

    Cerca e Sostituisci: Ctrl + H

    Vai alla riga n°: Ctrl + L

    Compila/Esegui: F8 / F5

🎨 Grafica e Design
GIMP (Fotoritocco)
Strumento   Scorciatoia
Sposta  M
Ritaglio    Shift + C
Testo   T
Zoom automatico Ctrl + Shift + J
Livelli Ctrl + L
Esporta Ctrl + Shift + E
Inkscape (Vettoriale)
Strumento   Scorciatoia
Seleziona/Sposta    S o F1
Modifica Nodi   N o F2
Crea Rettangolo R o F4
Crea Cerchio    E o F5
Raggruppa oggetti   Ctrl + G
Separa oggetti  Ctrl + Shift + G

🛠 Scorciatoie di Sistema MX Linux (XFCE)

    Apri Menu Applicazioni: Alt + F1 (o tasto Super/Windows)

    Passa da una finestra all'altra: Alt + Tab

    Chiudi finestra corrente: Alt + F4

    Cattura Schermata (Screenshot): Stamp (Print Screen)

    Blocca Schermo: Ctrl + Alt + L

💡 Suggerimento per Geany

Dato che usi Geany spesso, ricorda che puoi trascinare un file di testo direttamente sull'icona di Geany nella barra delle applicazioni per aprirlo istantaneamente.
