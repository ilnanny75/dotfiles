---

### 🚀 Setup Istantaneo
cd ~/dotfiles && chmod +x ilnanny-OS-manager.sh && ./ilnanny-OS-manager.sh

### 📂 Struttura Modulare
* bash/etc_bash/bashrc.d: Moduli Bash Universali
* Debian / Void / Arch: Configurazioni Specifiche
* scripts/bin: Script di automazione
* config: Config per Geany, Openbox, Thunar e XFCE

### 📔 MEMO Rapido
* pigia: Invia su GitHub
* up: Sincronizza locale
* instally: Installa pacchetti
* update: Aggiorna sistema
* treed: Struttura cartelle
